twitlyr by chad3814

a set of tools to upload and download files from twitter

I start this on a lark to see if one could (ab)use twitter this way. Mostly completely functioning, twitlyr.sh takes three arguments, the file name you want to upload and the username and password of the use to do the tweeting. detwitlyr.sh only takes one argument, the username from whom the first file it finds in the tweet-stream it will download.
